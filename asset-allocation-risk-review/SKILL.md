# 资产配置风险复盘

## 核心定位

把家庭、个人或小企业的资金安排做成风险复盘表，重点看现金流安全、资产集中度、负债压力、流动性和极端情况承受能力。这个 skill 不推荐具体证券或买卖操作。

## 触发场景

- 用户想复盘自己的资产结构，但不想进入炒股建议
- 用户需要判断现金、应收、负债、保险、固定资产和长期投入是否失衡
- 用户想做家庭年度财务体检或企业资金安全检查
- 用户需要把复杂资产情况整理成一张风险表

## 工作流

### Step 1: 收集资产负债轮廓

按大类收集，不要求用户暴露敏感细节。可以用区间和比例替代精确金额：

```text
现金/活期资金：
短期应收或待收款：
长期投入：
固定资产：
负债与月供：
保险和保障：
未来 6-12 个月大额支出：
```

### Step 2: 识别结构性风险

检查五个维度：

```text
现金流：能否覆盖 6-12 个月必要支出
集中度：是否过度依赖单一资产、客户、平台或收入来源
期限错配：短钱是否投到长周期事项
负债压力：固定还款是否挤压经营或生活安全垫
保障缺口：重大风险发生时是否会拖垮现金流
```

### Step 3: 做压力测试

设计 2-3 个极端但现实的场景，例如收入下降、客户延期付款、利率上升、突发医疗、项目回款失败。计算会先出问题的环节。

### Step 4: 输出复盘和动作

只输出风险分级、调整原则和核查动作，不推荐具体股票、基金、币种或入场时点。

## 输出格式

```text
复盘对象：个人 / 家庭 / 小企业
隐私处理：本复盘可使用区间和比例，不需要精确账户信息
资产负债摘要：...
主要风险：
1. ...
2. ...
压力测试：...
优先修正项：...
建议补充的数据：...
边界声明：不提供具体证券买卖建议
```

## Gotchas

1. **不要要求用户暴露隐私**
   解释：优先使用区间、比例和匿名类别，不索要账号、身份证、家庭成员姓名等信息。

2. **不要变成荐股工具**
   解释：可以谈集中度、流动性和风险承受能力，不能给具体证券交易动作。

3. **不要忽略负债**
   解释：资产配置不是只看资产，固定还款、担保、应收账款和未来支出同样重要。

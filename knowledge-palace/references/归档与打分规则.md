## 打分系统

### 分数标准

| 分数范围 | 星级 | 含义 | 是否纳入核心认知 |
|---------|------|------|-----------------|
| 100 | ⭐⭐⭐ | 核心战略、立即重要、经验验证、可直接执行 | ✅ 是 |
| 95-98 | ⭐⭐⭐ | 重要战略、已验证、高执行价值 | ✅ 是 |
| 90-92 | ⭐⭐ | 重要指导、有参考价值、需要调整 | ✅ 是 |
| 85-88 | ⭐ | 参考级、局部适用、需要结合场景 | ❌ 否 |
| <85 | 无 | 参考信息，不纳入核心认知 | ❌ 否 |

### 打分维度

1. **战略重要性** - 是否影响核心决策
2. **验证程度** - 是否经过实践验证
3. **执行价值** - 是否有明确可执行的动作
4. **时间敏感性** - 是否需要立即行动
5. **可复用性** - 能否复用于类似场景

## 常见场景判断

### 需要决策的问题类型

1. **观点冲突**: 新内容与现有认知矛盾
2. **全新思考**: 首次出现的战略方向
3. **细节补充**: 需要用户确认具体数字/时间
4. **打分争议**: 90分边界附近的判断

### 不纳入核心认知的情况

- 用户明确说"不重要"、"不用记"
- 分数低于90分
- 纯执行层面的临时信息
- 过时且已更新的内容

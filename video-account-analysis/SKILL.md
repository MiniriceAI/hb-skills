# 微信视频号数据分析

## 核心定位

读取用户提供的视频号后台数据截图，提取并计算核心指标，与行业基准对比，定位具体问题，输出针对性的内容优化建议——不是泛泛而谈，而是直接指向"改什么、怎么改"。

---

## 触发场景

- 用户上传视频号后台截图，说"帮我分析一下数据"
- 用户说"我的完播率很低，帮我找原因"
- 用户说"这周的数据和上周对比一下"
- 用户提供多条视频数据，说"哪条表现最好"

---

## 工作流

### 第一阶段：数据提取

1. **读取截图，识别可用数据字段**

   逐一识别以下字段（标记"无"表示截图中未显示）：

   | 字段 | 单位 | 说明 |
   |------|------|------|
   | 播放量 | 次 | 视频总播放次数 |
   | 完播率 | % | 看完视频 / 开始播放 |
   | 平均播放时长 | 秒 | 用于反推实际完播情况 |
   | 点赞数 | 个 | |
   | 评论数 | 条 | |
   | 分享数 | 次 | |
   | 收藏数 | 个 | |
   | 涨粉数 | 人 | 该视频带来的新关注 |
   | 主页访问量 | 次 | 从视频进入主页的次数 |

2. **处理多视频对比场景**
   - 若截图包含多条视频，逐条提取，建立表格
   - 若截图模糊或数字无法辨认，标注该字段为"识别失败"，告知用户

### 第二阶段：计算派生指标

3. **计算以下派生指标**（有原始数据时必须计算）

   互动率 = (点赞 + 评论 + 分享 + 收藏) / 播放量 × 100%

   粉丝转化率 = 涨粉数 / 播放量 × 100%

   评论率（细分互动质量）= 评论数 / 播放量 × 100%

   分享率（传播力指标）= 分享数 / 播放量 × 100%

   主页转化率（内容吸引力）= 主页访问量 / 播放量 × 100%

### 第三阶段：行业基准对比

4. **对每个指标标注等级**

   | 指标 | 及格线 | 良好 | 优秀 |
   |------|--------|------|------|
   | 完播率 | >15% | >25% | >35% |
   | 互动率 | >1% | >3% | >5% |
   | 粉丝转化率 | >0.1% | >0.5% | >1% |
   | 评论率 | >0.1% | >0.3% | >0.5% |
   | 分享率 | >0.1% | >0.5% | >1% |
   | 主页转化率 | >0.5% | >1.5% | >3% |

   备注：以上基准适用于粉丝量 1 万以下账号。粉丝量越大，各指标基准值通常越低。

5. **多视频场景：识别头部和尾部**
   - 按综合得分排序（完播率 × 40% + 互动率 × 40% + 粉丝转化率 × 20%）
   - 标记 Top 1-2 和 Bottom 1-2
   - 对比头部和尾部的差异特征

### 第四阶段：诊断决策树

6. **按以下顺序逐项诊断，找到第一个命中的问题**

   IF 完播率 < 15%
     → 问题在"开头钩子"
     → 检查：前3秒是否有悬念/冲突/利益承诺？
     → 检查：画面第一帧是否有视觉吸引力？
     → 建议：用问题式开头（"你知道为什么……"）或结果前置（先展示结果再解释）

   ELIF 完播率 15%-25% 但互动率 < 1%
     → 问题在"内容价值感"或"行动号召（CTA）"
     → 检查：视频结尾是否有明确的点赞/评论引导？
     → 检查：内容是否给观众留下了"想说点什么"的感受？
     → 建议：结尾加"你觉得……评论区告诉我"；内容中加入争议性观点

   ELIF 互动率 > 3% 但粉丝转化率 < 0.1%
     → 问题在"账号人设不清晰"
     → 用户喜欢这条视频但不知道关注你能获得什么
     → 建议：视频中明确说出"关注我，每周分享……"；完善主页简介

   ELIF 分享率 < 0.1%
     → 内容"自用"属性强，传播属性弱
     → 检查：内容是否有"值得发给朋友"的实用性或情绪共鸣？
     → 建议：加入"转发给需要的朋友"话术；制作工具型内容（模板/清单/公式）

   ELIF 播放量低（<1000）且其他指标正常
     → 问题在"分发阶段"而非内容质量
     → 可能原因：发布时间非高峰期（推荐 7-9点 / 12-13点 / 20-22点）
     → 可能原因：封面图文字过小或主题不够吸眼
     → 建议：调整发布时间；A/B测试封面

### 第五阶段：输出建议

7. 每条建议必须包含：
   - 具体问题是什么（数据支撑）
   - 根本原因（诊断结论）
   - 可操作的改进动作（能立即执行的）

---

## 输出格式

```
## 数据概览

| 指标 | 数值 | 行业基准 | 评级 |
|------|------|----------|------|
| 完播率 | 22% | >25%良好 | 待提升 |
| 互动率 | 4.2% | >3%良好 | 良好 |
| 粉丝转化率 | 0.3% | >0.1%及格 | 及格 |

## 诊断结论

核心问题：[一句话说明最主要的问题]

## 优先改进项（按影响力排序）

### 1. [问题名称]
- 数据表现：完播率 22%，低于良好基准（25%）
- 根本原因：[诊断]
- 改进动作：[具体可操作的 1-2 步]

### 2. [问题名称]

## 保持不变的优势
- [当前表现好的指标及内容策略，告知用户继续保持]

## 本周 vs 上周（仅在提供对比数据时）
[对比表格 + 变化趋势]
```

## Gotchas

### 截图数字识别模糊
- 若数字无法确认（如 1K 还是 1W），明确告知用户"X字段识别不确定，读取为[值]，请确认"
- 不要猜测，猜错会导致分析结论完全错误

### 完播率异常高（>80%）
- 可能原因：视频时长极短（<15秒），完播率天然高，不代表内容质量好
- 分析时需结合视频时长说明

### 数据量太少（播放量 < 500）
- 样本量不足，指标波动大，不具统计意义
- 说明"当前播放量偏低，数据仅供参考，建议积累 1000+ 播放后再做分析"

### 行业差异
- 上述基准适用于泛内容账号
- 垂直行业账号（教育、财经、医疗）互动率通常偏低（0.5-1.5% 属正常）
- 若用户提供了账号类型，调整基准并说明

### 多张截图时间跨度不一致
- 若截图来自不同时间段，提醒用户数据横向对比可能不公平
- 建议用同一时间窗口的数据进行比较

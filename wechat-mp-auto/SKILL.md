# 公众号自动化

## 核心定位

端到端处理微信公众号图文发布：从接收主题到在公众号编辑器中完成排版、预览并发布。内容生成遵循微信平台规范，操作通过浏览器自动化完成。

---

## 触发场景

- 用户说："帮我写一篇公众号文章，主题是……"
- 用户提供了草稿，说"帮我发到公众号"
- 定期内容发布任务中的执行节点
- 用户说："写完后帮我排版发出去"

---

## 工作流

### 第一阶段：内容需求确认

1. **收集必要信息**（若用户未提供，逐项询问）
   - 主题 / 关键词
   - 目标读者（普通用户 / 行业从业者 / 学生等）
   - 期望字数（短文 800-1200 字 / 中文 1500-2500 字 / 长文 3000+ 字）
   - 文章风格（知识科普 / 故事叙述 / 干货清单 / 观点评论）
   - 是否有封面图（若无，说明需手动处理）
   - 发布时间（立即 / 定时）

### 第二阶段：内容生成

2. **生成标题**
   - 主标题：≤ 64 个字符，符合以下任一结构：
     - 数字型："5个方法让你……"
     - 问题型："为什么你……"
     - 利益型："学会这一点，……"
     - 故事型："我……之后，才明白……"
   - 同时生成 3 个备选标题，供用户选择
   - 禁止使用夸张绝对化表述（"史上最"、"没有之一"、"颠覆"）

3. **生成摘要（abstract）**
   - ≤ 120 个字符
   - 概括文章核心价值，不是标题的重复
   - 将出现在公众号文章列表摘要位置和分享卡片

4. **生成正文结构**
   - 开头（100-150字）：抛出问题或场景，建立共鸣
   - 正文段落：每 300-500 字设一个 H2 小标题
   - 小标题数量：800字文章 2-3 个；2000字文章 4-6 个
   - 结尾（100-200字）：总结 + 行动号召（留言、分享、关注）
   - 段落长度：每段 ≤ 150 字，避免大段文字墙

5. **微信专属格式规范**
   - 不含任何外部链接（http/https 链接在正文中完全禁止，会被屏蔽）
   - 不含其他平台名称（抖音、微博、B站等），如需提及用"某平台"替代
   - 图片描述用 [图片] 占位，标注图片说明（实际图片需单独上传）
   - 可使用 emoji 但不要每句都有，控制在全文 5-8 个
   - 引用数据必须有来源说明（即使无链接也写"来源：XXX"）

6. **禁止内容检查**（生成后自动过滤）
   - 涉及政治敏感话题
   - 医疗功效绝对化表述
   - 金融收益承诺
   - 赌博、成人内容相关词汇
   - 以上内容如果是用户需求主题，明确告知无法处理

### 第三阶段：进入微信公众号编辑器

7. **导航至编辑器**
   - 打开 https://mp.weixin.qq.com
   - 检测是否已登录（查找"新建图文"或"写文章"按钮）
   - 若未登录 → 停止，提示用户扫码登录后继续

8. **新建图文**
   - 点击"写文章"或"新建图文消息"

9. **填写标题**
   - 点击标题输入框
   - 输入选定标题

10. **粘贴正文**
    - 点击正文编辑区
    - 使用编辑器的"粘贴为纯文本"功能（避免携带外部格式）
    - 逐段检查格式是否正确渲染

11. **设置 H2 小标题格式**
    - 选中每个小标题文字
    - 应用"标题2"样式（公众号编辑器中为字体大小或样式选项）

12. **处理图片**
    - 若用户提供了图片文件：
      - 将光标定位到 [图片] 占位符处
      - 删除占位符
      - 点击"插入图片"→ 上传对应图片
      - 等待图片上传完成
    - 若未提供图片：保留文字说明，告知用户需手动插入

13. **设置封面图和摘要**
    - 找到"封面和摘要"设置区域
    - 封面图：若用户提供则上传；否则从正文图片中选第一张
    - 摘要：粘贴第2步生成的 120 字摘要

14. **手机预览**
    - 点击"预览"→ 输入微信号（若需要）发送到手机
    - 或使用编辑器内的手机预览窗口
    - 检查要点：标题是否被截断、图片是否正常、段落排版是否美观

### 第四阶段：发布

15. **选择发布方式**
    - 立即发布：点击"发表"→ 确认弹窗 → 点击"确定"
    - 定时发布：点击"定时"→ 选择日期时间 → 点击"确定"

16. **确认发布成功**
    - 检测页面出现"发表成功"提示
    - 记录文章链接（在内容管理页面可获取）

## 输出格式

**内容生成完成（等待确认发布）**
```
文章已生成，请确认：

标题（已选）：[标题]
备选标题：
  A. [备选1]
  B. [备选2]
  C. [备选3]

摘要：[120字摘要]

正文结构：
  - 开头
  - H2：[小标题1]
  - H2：[小标题2]
  - 结尾

字数：约 [N] 字
图片占位：[N] 处
```

**发布成功**
```
发布成功
- 文章标题：[标题]
- 发布时间：[时间]
- 文章链接：[链接（若可获取）]
```

**发布失败**
```
发布失败
- 原因：[具体原因]
- 手动操作：
  1. 登录 https://mp.weixin.qq.com
  2. 点击"写文章"
  3. [后续步骤]
- 已生成内容：[提供可直接复制的标题和正文]
```

## Gotchas

### 外部链接导致审核失败
- 症状：文章发布后显示"内容违规"或无法打开
- 预防：生成内容时彻底扫描 http/https 字符串，一律删除或替换为文字说明

### 编辑器粘贴携带 HTML 格式
- 症状：正文出现乱码、奇怪符号或意外加粗
- 处理：使用"清除格式"按钮 → 重新粘贴 → 手动重设标题样式

### 封面图比例问题
- 微信封面图推荐比例：2.35:1（900×383px）或 1:1（正方形）
- 若用户图片比例不符，在输出中说明，建议裁剪后重新上传

### 定时发布时区
- 公众号定时发布使用北京时间（UTC+8）
- 确认用户输入的时间是否为北京时间，若不明确则询问

### 文章发出后无法修改正文
- 微信公众号图文一旦群发无法修改正文内容
- 仅可修改标题和原创声明
- 发布前务必完成手机预览确认

### 每日群发限制
- 认证账号：每日最多 1 篇群发
- 未认证账号：每月最多 4 篇
- 若已超出限额，操作将失败，在输出中提示用户次日再发

### 嵌入图片的 CDN 地址
- 公众号编辑器上传的图片会转为微信 CDN 地址
- 这些地址在微信外部无法访问，属于正常现象
- 不要将 CDN 图片链接当作可分享的图片链接使用

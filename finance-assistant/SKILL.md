# 核心定位

专为知识付费 / 在线教育业务设计的财务分析工具。核心任务是把原始收支数据转化为可操作的经营洞察：哪个渠道 ROI 最高、获客成本是否合理、利润结构是否健康，以及与同类业务的基准对比。

---

# 触发场景

- 用户提供月度/季度收支数据，需要整理分析
- 用户问"我的获客成本合理吗"、"哪个投放渠道最划算"
- 用户需要生成财务报告给投资人或合伙人
- 用户不确定某笔费用应该怎么归类

---

# 知识付费业务财务特殊性

与传统电商/SaaS 的关键区别：
- 收入确认：预收款（课程费用收到）不等于已实现收入（需按授课进度确认）
- 高退款率：7日无理由退款是标配，必须从收入中扣除
- 平台抽成：微信/抖音/小红书等各平台有不同抽成比例，影响净收入
- 分销佣金：私域/KOL 分销的佣金支出应计入 CAC，不能漏
- 内容资产摊销：课程录制成本属于资本性支出，不应全部计入当期费用

# 工作流

## Step 1: 数据收集

向用户索取或让用户填写以下数据表。必填项缺失则暂停分析并明确说明原因。

### 收入端（必填）
```
毛收款（Gross Revenue）：[金额]  ← 平台显示的销售总额
退款金额：[金额]
平台手续费/技术服务费：[金额]   ← 微信支付约0.6%，第三方平台更高
分销佣金支出：[金额]
```

### 成本端（必填）
```
广告投放费用（按渠道分列）：
  · 抖音投流：[金额]
  · 朋友圈广告：[金额]
  · 小红书投放：[金额]
  · 百度/SEM：[金额]
  · KOL/KOC 合作费：[金额]
  · 其他渠道：[金额]

内容/交付成本：
  · 讲师课酬：[金额]（内部员工薪资按比例分摊）
  · 内容制作（录音/剪辑/设计）：[金额]
  · 直播技术/设备折旧：[金额]

运营成本：
  · 工具订阅（企业微信服务商/CRM/数据工具等）：[金额]
  · 客服人力（按比例分摊）：[金额]
  · 社群运营人力：[金额]

管理费用（可选，用于完整P&L）：
  · 办公室租金分摊：[金额]
  · 管理层薪资分摊：[金额]
```

### 用户数据（计算 CAC/LTV 必需）
```
本期新增付费用户数：[人数]
本期复购用户数：[人数]
总付费用户基数：[人数]
平均客单价：[金额]（或分产品线提供）
用户平均生命周期内购买次数（若有历史数据）：[X次]
退款率（退款人数/付款人数）：[X%]
```

## Step 2: 数据清洗与口径统一

**必须在计算前完成以下检查：**

```
检查1：确认"毛收款"是否包含了退款中的金额
        → 净收入 = 毛收款 - 退款 - 平台手续费

检查2：分销佣金是否已从收入端扣除？
        → 建议：佣金计入 CAC（成本），不从收入扣减
        → 避免同时扣减导致双重计算

检查3：内容制作费是一次性还是可复用？
        → 可复用内容（录播课）= 资本性支出，建议按12-24个月摊销
        → 直播/活动性内容 = 当期费用，全额计入

检查4：人力成本是否混入了多个业务线？
        → 若团队同时服务多个产品，按收入占比分摊

检查5：广告投放数据的时间口径
        → 投放费用发生时间 vs 转化订单时间可能跨期
        → 建议：以付款时间对齐，若有明显跨期则说明
```

## Step 3: 核心指标计算

### 收入层

```
净收入（Net Revenue）= 毛收款 - 退款 - 平台手续费
净收入率 = 净收入 / 毛收款

毛利润 = 净收入 - 内容/交付成本
毛利率 = 毛利润 / 净收入
```

**知识付费毛利率基准：**
```
录播课为主：毛利率 60-80%（内容边际成本低）
直播课为主：毛利率 40-65%（讲师成本高）
1v1辅导：  毛利率 20-45%（人力密集）
```

### CAC（用户获取成本）

```
总获客费用 = 广告投放费用总和 + KOL合作费 + 分销佣金
CAC = 总获客费用 / 本期新增付费用户数

按渠道 CAC：
  渠道A CAC = 渠道A投放费用 / 渠道A带来的新增付费用户数
```

分销佣金必须计入 CAC，不能遗漏。

### LTV（用户生命周期价值）

```
方法一（有历史数据）：
  LTV = 平均客单价 × 平均购买次数 × 毛利率

方法二（无历史数据，估算）：
  LTV = 平均客单价 × (1 + 复购率估算) × 毛利率
  → 复购率参考：私域运营强的业务 30-50%，纯广告流量 10-20%

方法三（简化版，适合初创）：
  LTV = 当前平均客单价 × 毛利率
  → 保守估算，忽略复购，适合无历史数据时
```

### LTV:CAC 比率与回收期

```
LTV:CAC = LTV / CAC

CAC 回收期（月）= CAC / (平均客单价 × 毛利率)
```

**知识付费行业基准：**
```
LTV:CAC ≥ 3x         → 合格
LTV:CAC ≥ 5x         → 良好，可加大投放
LTV:CAC < 2x         → 警戒，需优化
LTV:CAC < 1x         → 危险，每新增一个用户都在亏损

CAC 回收期 ≤ 3个月   → 优秀，现金流健康
CAC 回收期 4-6个月   → 可接受，需监控
CAC 回收期 > 6个月   → 资金压力大，谨慎扩张
```

### 渠道 ROI

```
渠道 ROI  = 渠道带来的净收入 / 渠道投放费用
渠道 ROAS = 渠道带来的毛收款 / 渠道广告费用

ROI 用净收入（已扣成本），评估实际盈利性
ROAS 用毛收款（评估广告本身效率用）
```

### 完整 P&L

```
毛收款
- 退款
- 平台手续费
= 净收入（Net Revenue）

- 内容/交付成本
= 毛利润（Gross Profit）

- 市场推广费（广告+KOL+分销佣金）
- 运营人力费
- 工具订阅费
= 经营利润（Operating Profit / EBIT）

- 管理费用
= 净利润（Net Profit）
```

## Step 4: 对比分析

**和上期对比（若有历史数据）：**
```
对比维度：净收入、毛利率、CAC、LTV:CAC、各渠道ROI
计算环比变化（%）
重点标注：毛利率下降 / CAC上升 / 某渠道ROI大幅变化
```

**和目标对比（若有目标）：**
```
达成率 = 实际值 / 目标值
标注超额完成（>110%）和未达标（<90%）的指标
```

## Step 5: 渠道 ROI 表与建议

生成渠道对比表，并给出扩张/缩减建议。

扩张建议条件：渠道ROI > 3x 且带来用户质量高（复购率高于均值）

缩减建议条件：渠道ROI < 1.5x 或 CAC回收期 > 6个月

持观察建议条件：数据量不足（该渠道新增用户 < 50 人）

# Gotchas

**收入口径混淆（最常见错误）**
- 毛收款（平台流水）不等于净收入：不扣退款/手续费就做分析，会高估盈利能力
- 分期付款业务：按合同总额确认还是按收到款确认，要统一口径
- 赠品课/体验课：若收费为 1 元或 0 元，不应计入正常客单价计算 LTV

**CAC 漏算项**
- 分销佣金最容易被遗漏（有时被归入"推广物料"或"销售提成"，看内容归类）
- 内容营销人力（写推文、做短视频的员工时间）若较大，也应部分计入 CAC
- 老带新红包/优惠券：属于 CAC，不是折扣

**LTV 高估陷阱**
- 新业务（成立 < 1 年）没有足够的复购数据，用历史复购率估算 LTV 不可靠
- 应明确告知 LTV 是估算，并说明假设
- 高退款率会压低 LTV，退款率 > 15% 是重要警告信号

**P&L 结构混淆**
- 内容制作一次性大投入（如录制一套课程花了 10 万元）不应全计入当月费用——应摊销
- 预收款（报名费）不是收入——未授课的部分在会计上是负债（预收账款）
- 若业务有多个产品线，分别核算比合并核算更有价值

**基准使用注意**
- 基准数据适用于成熟期业务（月收入 > 50 万）；早期业务数据量小，CAC 波动大，基准对比意义有限
- 不同获客渠道的 LTV 差异可能很大（私域用户 LTV 通常是广告流量的 2-3 倍），合并计算会掩盖差异

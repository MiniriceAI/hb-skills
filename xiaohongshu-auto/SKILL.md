# 自动发小红书

## 核心定位

自动化执行小红书图文笔记发布流程。接收用户提供的标题、正文、图片和可选时间，完成从内容校验到发布确认的全链路操作，失败时提供可操作的手动回退指引。

---

## 触发场景

- 用户说："帮我发到小红书"
- 用户提供了标题 + 正文 + 图片路径列表
- 用户说："发笔记，定时明天上午 9 点"
- 批量发布任务中的单条执行节点

---

## 工作流

### 第一阶段：内容验证（发布前必须全部通过）

1. **标题校验**
   - 长度 ≤ 20 个字符（含标点、emoji）
   - 不含外部链接（http/https/www）
   - 不含联系方式（手机号、微信号、QQ）
   - 超长时，截取前 17 字符 + "..."，告知用户

2. **图片校验**
   - 数量：1 至 18 张（含）
   - 格式：JPG / PNG / WEBP（不支持 GIF 动图作为普通图片）
   - 单张大小：建议 ≤ 20MB
   - 如有格式不符，列出问题文件，中止流程，告知用户转换后重试

3. **正文内容校验**
   - 不含外部链接（小红书会检测并屏蔽）
   - 不含竞品平台名称（抖音、B站、微博等）出现时标记风险，提示用户确认
   - 字数无硬性上限，但建议 ≤ 1000 字以保证阅读体验

4. **敏感词预检**
   - 检查以下高风险词类：医疗承诺类（"根治"、"特效"、"100%有效"）、金融收益类（"稳赚"、"保本"）、虚假宣传类（"最"、"第一"、"唯一"）
   - 发现时高亮标出，询问用户是否替换或继续

### 第二阶段：生成话题标签

5. 基于正文内容，生成 3-5 个话题标签
   - 规则：每个标签以 # 开头，标签词 2-10 个字，不含空格
   - 优先选用有搜索热度的通用词（行业词 > 细分词 > 品牌词）
   - 示例：正文为护肤教程 → #护肤 #敏感肌护肤 #护肤技巧 #素颜感
   - 将标签追加到正文末尾（换行隔开）

### 第三阶段：浏览器操作

6. **打开小红书创作平台**
   - 导航至 https://creator.xiaohongshu.com/publish/publish
   - 等待页面加载完成（检测"发布笔记"按钮可见）
   - 若检测到登录跳转 → 执行【登录过期处理】（见 Gotchas）

7. **点击"发布笔记"**
   - 选择"图文笔记"（非视频）

8. **上传图片**
   - 按用户提供的顺序逐张上传
   - 等待每张图片上传进度条消失后再上传下一张
   - 上传完成后校验缩略图数量与输入一致

9. **填写标题**
   - 点击标题输入框
   - 粘贴处理后的标题（已截断或原始）

10. **填写正文**
    - 点击正文输入框
    - 粘贴正文 + 话题标签

11. **添加话题**
    - 若平台提供独立话题输入框，在此处再次输入话题（部分版本需双重添加）

12. **设置发布时间**
    - 立即发布：选择"立即发布"
    - 定时发布：点击"定时发布" → 选择日期和时间 → 确认（定时时间需为当前时间 2 小时后）

13. **点击发布 / 确认**
    - 点击"发布"按钮
    - 等待成功提示（通常为"发布成功"弹窗或跳转至内容管理页）

### 第四阶段：发布确认

14. **验证发布成功**
    - 方式一：检测页面出现"发布成功"文字
    - 方式二：导航至 https://creator.xiaohongshu.com/creator/post，检查最新一条内容标题与发布标题一致
    - 记录笔记 ID（URL 中的 note_id）

15. **输出结果**
    - 成功：返回笔记链接、发布时间、话题标签列表
    - 失败：返回错误截图描述 + 手动回退指引

---

## 输出格式

**发布成功**
```
发布成功
- 标题：[实际发布的标题]
- 笔记链接：https://www.xiaohongshu.com/explore/[note_id]
- 发布时间：[时间]
- 使用话题：[话题1] [话题2] [话题3]
- 图片数量：[N] 张
```

**发布失败**
```
发布失败
- 失败原因：[具体原因]
- 失败步骤：[第N步]
- 手动操作指引：
  1. 打开 https://creator.xiaohongshu.com/publish/publish
  2. [具体操作步骤]
- 已准备好的内容（可直接复制）：
  标题：[标题]
  正文：[正文+话题]
```

## Gotchas

### 登录过期
- 症状：页面跳转至 https://www.xiaohongshu.com/login
- 处理：停止自动化 → 告知用户"检测到登录状态失效，请手动扫码登录后告知我，我将继续操作"
- 不要尝试自动填写账号密码

### 图片格式不支持
- 症状：上传后图片显示感叹号或上传进度卡住
- 处理：记录问题图片序号 → 提示用户将该图片转换为 JPG 格式后重新提供

### 敏感词拦截
- 症状：点击发布后出现"内容包含违规词"提示
- 处理：截取提示文字 → 返回正文 → 标记可疑词 → 提供替换建议 → 请用户确认后重试

### 定时时间不合法
- 症状：时间选择器报错或无法选中
- 原因：定时时间 < 当前时间 + 2 小时
- 处理：自动将时间调整为当前时间 + 2.5 小时，告知用户

### 平台功能改版
- 症状：找不到"发布笔记"按钮或页面结构异常
- 处理：截取当前页面截图描述 → 提示用户小红书界面可能已更新 → 提供当前页面元素描述请用户指引

### 发布后内容审核中
- 症状：发布成功但内容显示"审核中"
- 这是正常状态，审核通常 1-24 小时完成
- 在输出中注明，不视为失败
